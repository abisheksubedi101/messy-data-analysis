# messy-data-analysis
## Project Overview
This project analyzes sales data to identify trends, top-performing products, and regional performance. The goal is to provide insights that can help improve business decisions.

## Business Problem
The company wants to understand:
- Which products generate the most revenue?
- What are the top ten sold products?
- Which day of the month have generated most revenue?

---

## Dataset
- Source: Kaggle
- Records: 100+
- Features:
  Sales_ID
  Date
  Custumer_Name
  Product	Category
  Quantity
  Unit_Price
  Total
  Payment_Method


---

## Tools Used
- SQL
- Microsoft Excel
- Power BI
- python for cleaning

---

## Project Workflow
1. Imported the dataset.
2. Cleaned missing and duplicate values.
3. Performed exploratory data analysis.
4. Wrote SQL queries to answer business questions.
5. Created KPIs and visualizations.
6. Built an interactive Power BI dashboard.
